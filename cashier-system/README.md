# 收银系统 (Cashier System)

## 项目结构

```
cashier-system/
├── src/
│   ├── pages/          # 页面 HTML
│   │   ├── login.html  # 登录页
│   │   └── ...
│   ├── assets/
│   │   ├── css/        # 样式文件

│   │   └── images/     # 图片资源
│   └── config/         # 配置文件
```
