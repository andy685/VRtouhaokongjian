## 支付流程说明

```mermaid
flowchart TD
    A[扫描会员码] --> B{识别成功?}
    B -->|是| C[显示待支付项目]
    B -->|否| D[扫描失败提示]
    D --> E[重试/手动输入]
    E --> A
    C --> F[选择支付方式]
    F --> G[确认支付]
    G --> H[支付处理中]
    H --> I{支付成功?}
    I -->|是| J[支付成功页]
    I -->|否| K[支付失败提示]
    K --> F
    J --> L[分配设备 → 开始体验]
```

## 交互状态表

| 状态 | 核心元素 | 按钮文案 | 备注 |
|------|---------|---------|------|
| **待支付** | 会员信息卡 + 商品明细 + 支付方式选择 | `确认支付 ¥36.10` | 默认选中余额支付 |
| **支付中** | 加载动画 + 金额显示 | - | 不可操作，约 1-3 秒 |
| **支付成功** | 成功图标 + 订单摘要 + 设备引导 | `返回收银台` | 显示剩余余额 |
| **扫码失败** | 失败图标 + 原因说明 | `重新扫描` / `手动输入会员号` | 两种应急方案 |
